# RTSPServer

## 项目介绍

一个RTP OVER RTSP/UDP的RTSP服务器

## 功能介绍

- 此项目默认采用RTP_OVER_UDP
- 支持使用H264格式的视频文件作为媒体源
- 其它功能有时间本人会不断完善

## 开发环境

- gcc version 9.3.0 (Ubuntu 9.3.0-17ubuntu1~20.04) 
- Code::Blocks 20.03
- C++

## 技术

- 技术简介

  RTSP消息处理：采用单Reactor模型，非阻塞IO，实现方式参考了redis的Reactor模型
  RTP数据发送：使用工作线程来处理数据量比较大的RTP数据发送和音视频文件处理任务，每一个媒体源对应一个工作线程

- IO多路复用

  暂时只支持epoll
